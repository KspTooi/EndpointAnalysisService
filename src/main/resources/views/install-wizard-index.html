EAS系统安装向导

欢迎使用 EAS 安装向导

这个向导将帮助您完成 EAS 的初始化设置或版本升级，确保系统能够正常运行。 安装过程简单快捷，只需几个步骤即可完成。

安装向导会在首次安装或系统版本升级时自动运行，确保系统数据和配置保持最新。
安装过程包括以下步骤：
1
系统介绍

了解 EAS系统的功能和特性，以及安装向导将完成的任务。

2
数据初始化

初始化系统必要的数据，包括权限节点、用户组、默认用户和全局配置等。

3
完成安装

确认所有设置并完成安装，准备开始使用 EAS。

系统要求
  • Java 21 或更高版本
  • 至少 2GB 可用内存
  • 至少 500MB 可用磁盘空间
  • 支持的操作系统：Windows 10/11, macOS, Linux
